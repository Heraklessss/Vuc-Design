HOW IT WORKS
────────────
This folder contains placeholder files that show you exactly
which image goes where on your website.

To add YOUR image: just replace the placeholder file with
your own image — using the EXACT SAME filename.

For example:
  • Delete  →  project-01.jpg  (the placeholder)
  • Add     →  project-01.jpg  (YOUR design image)
  That's it. Open index.html and it shows your image.

════════════════════════════════════════════════════
  TIPS FOR BEST QUALITY
════════════════════════════════════════════════════

  ✓ Convert images to WebP using squoosh.app (free, fast)
  ✓ Keep file size under 200KB per image for fast loading
  ✓ Use 16:9 ratio for consistent grid layout
  ✓ If using WebP, rename the file:
      project-01.jpg  →  project-01.webp
    Then update index.html: src="images/project-01.webp"
